# Projeto Volley Manager

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Marcelo20Torres/projeto_ser_front-end.git
   ```

2. **Acesse o diretório do projeto:**
   ```bash
   cd volley-system
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
